<!DOCTYPE> defines that this document is an HTML5 document
<meta> meta element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings. The metadata will not be displayed on the page, but is used by browsers (how to display content or reload page), by search engines (keywords), and other web services.
<meta name="author" content="John Doe"> Define the author of a page
<meta charset="UTF-8"> Define the character set used
<meta name="keywords" content="HTML, CSS, JavaScript"> Define keywords for search engines
<meta name="description" content="Free Web tutorials"> Define a description of your web page
<meta http-equiv="refresh" content="30"> Refresh document every 30 seconds
<meta name="viewport" content="width=device-width, initial-scale=1.0"> Setting the viewport to make your website look good on all devices. This gives the browser instructions on how to control the page's dimensions and scaling. The "width=device-width" part sets the width of the page to follow the screen-width of the device (which will vary depending on the device). The "initial-scale=1.0" part sets the initial zoom level when the page is first loaded by the browser. You can see the difference of webpage using viewport meta tag, and didn't in W3school.

<colgroup> This element is used to style specific columns of a table. element should be used as a container for the column specifications. this tag must be a child of a "table" element and should be placed before any other table elements, like "thead", "tr", "td" etc., but after the "caption" element, if present. There is only a very limited selection of CSS properties that are allowed to be used in the colgroup: "width property", "visibility property", "background properties", "border properties". All other CSS properties will have no effect on your tables.
<col> You'll use this element inside the colgroup tag. Each group is specified with this element. The span attribute specifies how many columns get the style. The style attribute specifies the style to give the columns. If you want to style multiple columns with different styles, use more than one "col" element inside the "colgroup". If you want to style columns in the middle of a table, insert an "empty" col element (with no styles) for the columns before. You can hide columns with the visibility: collapse property.

<th> stands for a table header cell in an HTML table. It is used to define headings for columns or rows within the table. Each th represents a table cell. these elements are typically bold and centered by default, providing semantic meaning to data and aiding accessibility for screen readers and other assistive technologies.

The id attribute is most used to point to a style in a style sheet, and by JavaScript (via the HTML DOM) to manipulate the element with the specific id.

I haven't review much about CSS but I'll describe what I've learned today. CSS stands for Cascading style sheets. We use it to style the HTML like change the text color, how the page look, and much more

How to link CSS and HTML

In head section of the html use link to link it. Inside the link use rel to tell the relation of the CSS document to HTML document. Use href to make a reference to the CSS file location. Last, use type to detail the type of document you've imported to the HTML file.

<link rel="stylesheet" href="the CSS file location" type="text/css>

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style. We can divide CSS selectors into five categories:

  • Simple selectors (select elements based on name, id, class).
  • Combinator selectors (select elements based on a specific relationship between them).
  • Pseudo-class selectors (select elements based on a certain state).
  • Pseudo-elements selectors (select and style a part of an element).
  • Attribute selectors (select elements based on an attribute or attribute value).
This section will explain the most basic CSS selectors.

The CSS element Selector element {
text-align: center;
color: red;
}
The CSS id Selector #id {
text-align: center;
color: red;
}
The CSS class Selector .class {
text-align: center;
color: red;
}
element.class {
text-align: center;
color: red;
}
The CSS Universal Selector * {
text-align: center;
color: red;
}
The CSS Grouping Selector element1, element2, element3 {
text-align: center;
color: red;
}
